运行示例

每个示例都包含完整的代码和运行说明。您可以直接复制代码并运行：

# 克隆示例代码
git clone https://github.com/your-username/ypython-examples.git
cd ypython-examples

# 安装依赖
pip install -r requirements.txt

# 运行示例
python examples/basic/hello-world.py
